# Kyro Reports

A web application for creating Excel templates and compiling multiple reports into a single document.

## Features

- **Template Creator**: Define fields and generate .xlsx templates
- **Compiler**: Upload a template and multiple reports, then compile them into a single file

## Workflow

### Template Creator
1. Enter template name
2. Add fields (dynamic - add as many as needed)
3. Download the .xlsx template

### Compiler
1. Upload template .xlsx file
2. Upload multiple report .xlsx files created from the template
3. Compile into a single document

## Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

## Build

```bash
npm run build
```
